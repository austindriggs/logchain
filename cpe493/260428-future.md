# Future Tasks or Recommendations

1. Some type of uptime-kuma, grafana, or prometheus monitoring system.
2. Custom python scripts that could scrape these logs, select keywords, and pass them along.
3. Some type of mesh network or VPN like Tailscale or reverse proxy like Nginx to allow outside access.
4. A notification and alerting system such as ntfy, pushover, gotify, or discord bot.
5. Some type of API or modularity to continue to allow future students to build on top.
