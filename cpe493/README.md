# CPE493 Hardware Security at West Virginia University

Files in this directory are related or required by the class.

This course will cover fundamentals of hardware security and potential solutions for common threats and vulnerabilities. Several applications will be explored as use cases, including Industrial Control Systems (ICS), Embedded Systems, and Cyber-Physical Systems (CPS), all of which will have associated hands-on exercise(s). Upon completing the course, we expect our students to understand the vulnerabilities in current digital systems and the physical attacks to these systems; to learn that security starts from hardware design and to be aware of the tools and necessary skills for building, testing, and operating secure and trusted hardware.
